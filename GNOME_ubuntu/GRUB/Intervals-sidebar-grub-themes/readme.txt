# Intervals Sidebar GRUB Themes

This folder contains a collection of GRUB themes designed for various display resolutions. Each theme comes with its own `install.sh` script for easy installation.

## Available Themes

- **1080p**: Recommended for most modern systems (1920x1080 resolution).
- **Other Resolutions**: Themes are also available for other common display sizes. Please choose the folder or script that matches your screen resolution.

## How to Install

1. Identify your display resolution.
2. Navigate to the corresponding folder or locate the appropriate `install.sh` script.
3. Open a terminal and run the installation script:
    ```sh
    chmod +x install.sh
    ./install.sh
    ```
4. Follow any on-screen instructions to complete the installation.

## Notes

- It is recommended to back up your existing GRUB configuration before installing a new theme.
- These themes are provided as-is. While the 1080p theme is widely compatible, results may vary depending on your system.
- If you encounter issues, you may need to adjust the theme or script for your specific setup.
